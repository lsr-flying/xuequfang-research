## 代码/文档规范

### 信息来源优先级
1. **官方文件** - 教育局官网、政府公告（最权威）
2. **官方解读** - 官方媒体的政策解读
3. **家长经验** - 篱笆、小红书等真实案例（需标注来源）
4. **中介信息** - 链家、中原等（参考，需交叉验证）

### 每条信息需标注
```yaml
updated: YYYY-MM    # 最后更新
source: 来源       # 信息来源
reliability: 高/中/低  # 可信度
```

### 研究原则
1. **时效性优先** - 政策每年变，必须用最新年份数据
2. **双向验证** - 重要信息至少两个来源确认
3. **标注时间** - 任何预测都要标注基于哪年数据
4. **保留原文** - 关键政策要保留原文引用

## 常用命令

### 运行研究任务
1. 使用 websearch 搜索最新政策
2. 使用 codesearch 搜索专业解读
3. 使用 webfetch 获取官方文件

### 工具推荐
- **浏览器**: 使用署名的MCP工具获取最新信息
- **画图**: 使用Mermaid生成流程图、时间线
