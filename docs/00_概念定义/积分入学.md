## 定义
> 积分入学是指外地户籍学生通过居住证积分达标获得公办学校入学资格

## 积分构成
| 项目 | 分值 | 说明 |
|------|------|------|
| 学历 | 50-110 | 本科90，硕士100 |
| 职称 | 100 | 中级职称 |
| 社保 | 每满1年+3 | 按月缴纳 |
| 居住证 | 每满1年+3 |  |
| 购房 | 80 | 自有产权房 |
| 社保基数 | 高于平均+更多 | |

## 达标线
- 上海: 120分（2024年）
- 嘉定: 120分

## 入学顺序
1. 有房+积分120 → 人户一致排前面
2. 租房+积分120 → 统筹
3. 有房+积分不足 → 统筹
4. 无积分 → 统筹或回家

## 办理流程
1. 居住证登记 → 6个月后领取
2. 积分申请 → 累计社保等
3. 入学申请 → 凭积分通知书

## 风险提示
- ⚠️ 积分要提前1-2年准备
- ⚠️ 社保持续不能断
- ⚠️ 购房+积分排序优先
