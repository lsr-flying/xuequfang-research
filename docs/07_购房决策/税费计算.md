## 购房税费汇总

### 新房税费

| 税费项目 | 计算方式 | 备注 |
|---------|----------|------|
| 契税 | 购房款×1%-3% | 首套90平以下1%，90平以上1.5%；二套3% |
| 维修基金 | 面积×单价 | 多层50元/平，高层80元/平 |
| 产权登记费 | 80元/件 | 住宅 |
| 抵押登记费 | 80元/件 | 贷款需支付 |

### 二手房税费

| 税费项目 | 计算方式 | 备注 |
|---------|----------|------|
| 契税 | 同新房 | |
| 个税 | 购房款×1%或20% | 满五唯一免征 |
| 增值税 | 购房款×5.3% | 满2年免征 |
| 中介费 | 购房款×1%-2% | 贝壳/链家标准 |

### 税费计算示例

**首套新房（90平以上，300万）**
- 契税: 300万×1.5% = 4.5万
- 维修基金: 90×80 = 0.72万
- 登记费: 80元
- **合计约: 5.2万**

**首套二手房（满五唯一，300万）**
- 契税: 300万×1.5% = 4.5万
- 个税: 免征
- 增值税: 免征
- 中介费: 300万×1% = 3万
- **合计约: 7.5万**

## 购房成本明细

### 总成本 = 房价 + 税费 + 中介费 + 其他

| 项目 | 说明 |
|------|------|
| 房价 | 合同价 |
| 税费 | 契税/个税/增值税等 |
| 中介费 | 二手房需支付 |
| 贷款费用 | 评估费/担保费等 |

---

## 注意事项

1. **满五唯一**: 个税和增值税免征，只付契税
2. **贷款成数**: 首套65%，二套50%（普通住宅）/40%（非普通）
3. **普通住宅标准**: 内环内≤450万，内外环间≤310万，外环外≤230万
