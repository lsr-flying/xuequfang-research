## 学校简介

菊园新区南片规划学校是嘉定区政府规划中的九年一贯制学校，旨在服务菊园新区南片新建住宅区居民的就学需求，缓解菊园地区学位压力。

## 基本信息

| 项目 | 内容 |
|------|------|
| **类型** | 公办九年一贯制 |
| **位置** | 菊园新区南片，胜竹路以南区域 |
| **状态** | 土地规划阶段 |
| **规模** | 约36个教学班（小学24班+初中12班） |
| **预计开工** | 2026年底-2027年初 |
| **预计招生** | 2028年9月（最乐观估计） |

## 2028年招生可行性评估

| 评估项 | 内容 |
|--------|------|
| **可行性** | 🟡 **中等** — 取决于2026-2027年开工进度 |
| **乐观情景** | 2026年底开工 → 2028年9月招生（概率40%） |
| **中性情景** | 2027年开工 → 2029年9月招生（概率40%） |
| **悲观情景** | 项目延后 → 2030年后（概率20%） |
| **关键节点** | 2026年是否列入区重大工程建设计划 |

### 对2028年入学的影响

| 情景 | 影响 |
|------|------|
| **2028年招生** | 菊园新区学校密度进一步提升，统筹距离更短；新校本身成为备选 |
| **2028年未招生** | 菊园现有学校（嘉定实验+中科院）竞争仍激烈，但统筹去向不变 |

## 建设背景

### 人口增长
- 菊园新区南片新建小区密集（书香嘉苑、领峯社区、蔷薇社区等）
- 入住人口持续增加
- 现有学校（嘉定实验小学、中科院实验学校）已红色预警

### 需求紧迫性
- 书香嘉苑、领峯社区等入住率提高
- 上海大学附属嘉定实验学校学位紧张
- 需要新学校分流缓解压力

## 对口小区（推测）

| 小区 | 现状对口 | 新校对口可能性 | 2026均价 |
|------|---------|--------------|---------|
| 书香嘉苑 | 上海大学附属嘉定实验 | 🔴 可能分流到此 | 4.2-4.8万 |
| 领峯社区 | 上海大学附属嘉定实验 | 🔴 可能分流到此 | 4.0-4.5万 |
| 蔷薇社区 | 上海大学附属嘉定实验 | 🟡 可能保留 | 3.8-4.3万 |
| 菊园新区南片新小区 | 暂未分配 | 🔴 大概率对口 | 待定 |

## 政策预测

### 五年一户
- 新学校预计首年不实施
- 逐年过渡

### 入户年限
- 首年可能无强制要求
- 视学位情况而定

## 发展潜力

### 优势
- 政府规划背书
- 九年一贯制省心
- 新校硬件一流
- 可能引入高校/名校资源（菊园高校聚集）

### 风险
- 尚在规划，进度不确定
- 学区划分可能有变化
- 师资待定
- 品牌效应未知

### 对菊园学区的影响
- 建成后嘉定实验小学和中科院实验学校的学位压力将缓解
- 嘉定实验小学的入户年限可能从满3年降至满2年
- 菊园整体统筹风险进一步降低

## 购房建议

### 风险提示
1. 规划可能调整，2028年招生不确定
2. 学区划分待官方确定
3. 建议关注2026年区重大工程名单

### 建议
- 购买前确认当前学区
- 如果看重确定性，优先嘉定实验小学和中科院实验学校现有对口小区
- 如果愿意冒风险赌新校，可关注南片新建小区
- 预留备选方案
