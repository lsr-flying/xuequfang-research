## 学校简介

上海市实验学校嘉定新城分校是由上海市教育委员会、上海师范大学、嘉定区人民政府三方合作共建，纳入上海市实验学校教育集团管理的高起点九年一贯制学校。

## 基本信息

| 项目 | 内容 |
|------|------|
| **建校时间** | 2024年9月正式启用（借址办学） |
| **新校舍** | 预计2027年9月启用 |
| **办学性质** | 公办九年一贯制 |
| **办学规模** | 54个教学班 |
| **校区地址** | 嘉定区马陆镇（新城核心区） |
| **首年招生** | 约300名新生 |
| **招生方式** | 电脑派位（区域内） |

## 招生范围

马陆镇"东至S5（沪嘉高速公路）、南至宝安公路、西至G15（沈海高速）、北至G1503（上海绕城高速）"的区域。

|------|-----------|---------|---------|-------|

## 2027年新校舍启用评估

| 评估项 | 内容 |
|--------|------|
| **确定性** | 🔴 **高** — 已列入区重大工程，2025年已开工建设 |
| **预计交付** | 2027年6月前 |
| **招生条件** | 新校舍启用后扩招可能性大（可能增至8-10班） |
| **对口调整** | 新校舍启用后可能调整学区划分，取消电脑派位改为对口入学 |
| **2028年入学** | 大概率已永久校址正常招生 |

### 2027-2028年入学预判

| 情景 | 概率 | 说明 |
|------|------|------|
| 维持电脑派位 | 40% | 沿用现有招生方式 |
| 改为对口入学 | 50% | 新校舍启用后划定固定学区（利好周边小区） |
| 电脑派位+部分对口 | 10% | 混合模式 |

### 对周边小区的影响

- **利好**: 新校舍周边小区（如骏丰嘉天下、新城金郡等）可能划入对口
- **风险**: 一旦改为对口入学，入户年限要求将快速提高
