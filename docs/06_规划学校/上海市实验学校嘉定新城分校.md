## 学校简介

上海市实验学校嘉定新城分校是由上海市教育委员会、上海师范大学、嘉定区人民政府三方合作共建，纳入上海市实验学校教育集团管理的高起点九年一贯制学校。

## 基本信息

| 项目 | 内容 |
|------|------|
| **建校时间** | 2024年9月正式启用 |
| **办学性质** | 公办九年一贯制 |
| **办学规模** | 54个教学班 |
| **校区地址** | 嘉定区马陆镇（新城核心区） |
| **首年招生** | 约300名新生 |
| **招生方式** | 电脑派位 |

## 招生范围

马陆镇"东至S5（沪嘉高速公路）、南至宝安公路、西至G15（沈海高速）、北至G1503（上海绕城高速）"的区域。

## 招生政策

### 电脑派位
- 区域内学生自愿报名
- 电脑随机录取
- 无需户籍年限

### 五年一户
- 暂未实施

### 入户年限
- 无强制要求

## 教学特色

### 背景
- 上海市实验学校教育集团
- 上实"充分发掘学生智慧潜能，充分发展学生个性特长"理念
- 综合改革实验性质

### 特色
- "改变设计，设计改变"办学特色
- 创新教育模式
- 综合素质培养
- 上实系课程体系

### 集团资源
- 依托上海师范大学
- 嘉定区政策支持
- 集团内教师培训
- 课程资源共享

## 潜力分析

### 优势
- 上实品牌（上海顶尖教育集团）
- 政府重点打造
- 硬件设施一流
- 无户籍预警压力（电脑派位）
- 集团化办学

### 风险
- 新建校成绩待验证
- 电脑派位不确定性
- 名额有限

### 发展预测
- 潜力巨大（上实系背景）
- 预计3-5年成为区顶尖

## 适合家庭

- 预算350-500万
- 在嘉定新城工作
- 可接受电脑派位
- 认可上实品牌
