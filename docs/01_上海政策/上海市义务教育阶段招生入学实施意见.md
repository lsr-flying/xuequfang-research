## 核心内容（2026年更新）

### 招生对象
- **小学**: 2019年9月1日-2020年8月31日出生，年满6周岁
- **初中**: 本市小学五年级学生

### 关键原则
1. **免试就近入学**: 公办学校按照免试原则
2. **公民办同步招生**: 公办民办同步报名
3. **阳光招生**: 公开透明，严禁选拔
4. **均衡分班**: 严禁举办重点班/实验班

### 民办学校招生
- 纳入审批地统一管理
- 与公办学校同步招生
- 报名超过计划→电脑随机录取
- 民办学校不得提前组织招生

### 时间安排（2026年）

| 时间 | 内容 |
|------|------|
| 4月10日 | 入学报名系统开通 |
| 4月12-13日 | 校园开放日 |
| 4月14-27日 | 入学信息登记 |
| 5月7-11日 | 小学入学网上报名 |
| 5月12-14日 | 民办初中网上报名 |
| 5月17-20日 | 公办小学第一批验证 |
| 5月21日 | 电脑派位/民办录取 |
| 8月15日前 | 发放入学通知书 |

### 重要提醒
- ⚠️ 严禁考试/竞赛/证书作为招生依据
- ⚠️ 严禁举办重点班/实验班
- ⚠️ 均衡分班，随机编班
- ⚠️ 严禁任何形式的选拔性考试

## 政策特点（2026年）
- **延续稳定**: 与2025年政策基本保持一致
- **数字化升级**: "教育入学一件事"系统进一步优化
- **监管加强**: 对违规招生行为加大查处力度
- **服务优化**: 一网通办服务更加便捷
