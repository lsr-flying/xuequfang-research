## 徐汇区学校概览

> **简评**：上海教育最强区之一（与浦东、闵行并称前三），但核心区房价极高且入学竞争惨烈。外围长桥/华泾板块有<7万的性价比选择。

### 一梯队小学

| 学校 | 特色 | 对口房价 | 录取难度 | 备注 |
|------|------|---------|---------|------|
| 世界小学 | 传统强校 | 10-15万 | 极高 | 核心区 |
| 汇师小学 | 传统强校 | 10-15万 | 极高 | 核心区 |
| 向阳小学 | 传统强校 | 10-15万 | 极高 | 核心区 |
| 建襄小学 | 传统强校 | 10-15万 | 极高 | 核心区 |
| 高安路第一小学 | 传统强校 | 10-15万 | 极高 | 核心区 |
| 上海小学 | 传统强校 | 6-8万 | 高 | 长桥板块 |
| 园南小学 | 传统强校 | 5-7万 | 中高 | 长桥/植物园 |

### 外围板块性价比选择

| 板块 | 小学 | 初中 | 房价 | 总价区间 |
|------|------|------|-----|---------|
| **长桥** | 上海小学/园南小学 | 园南中学/徐教院附中 | 5-7万 | 300-500万 |
| **华泾** | 华泾小学/徐浦小学 | 紫阳中学 | 5-6万 | 300-450万 |

### 一梯队初中

| 学校 | 中考市重点率 |
|------|------------|
| 华育中学(民办) | 约70%+ |
| 世外中学(民办) | 约50%+ |
| 西南位育(民办) | 约45%+ |
| 徐汇中学(公办) | 约30-35% |
| 位育初级(公办) | 约28-33% |
| 园南中学(公办) | 约25-30% |

### 2028年入学建议

| 预算 | 建议 |
|------|------|
| 300-450万 | 长桥板块（上海小学+园南中学），徐汇唯一性价比之选 |
| 450-600万 | 长桥/华泾，可考虑民办摇号 |
| 600万+ | 可进入核心区对口 |

**注意**：徐汇整体入学竞争全市最激烈之一，除长桥/华泾外，核心区入户年限要求极高（满3-5年）。
