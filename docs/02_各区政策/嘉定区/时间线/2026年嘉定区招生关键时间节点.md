## 家长行动清单（2026年）

### 提前准备（2025年9月-2026年3月）
- [ ] 了解2026年招生政策变化
- [ ] 确认对口学校和入户年限要求（满3年）
- [ ] 查询五年一户名额（2021年起计算）
- [ ] 准备入学材料（户口本、房产证等）
- [ ] 确认户籍迁入时间（2023年4月30日前）

### 信息收集（2026年4月）
- [ ] 参加校园开放日（4月12-13日）
- [ ] 实地考察对口小区
- [ ] 确认学校招生简章
- [ ] 完成入学信息登记（4月14-27日）

### 报名验证（2026年5月）
- [ ] 网上报名（公办/民办二选一）
- [ ] 准备验证材料原件及复印件
- [ ] 参加第一批验证（5月17-20日）
- [ ] 等待录取结果（5月21日公布）

### 入学准备（2026年6-8月）
- [ ] 领取入学通知书（8月15日前）
- [ ] 参加分班考/军训（如有）
- [ ] 购买学习用品
- [ ] 调整作息时间

## 重要提醒（2026年）

### 政策变化
- ⚠️ **入户年限**: 2026年须满**三年**（2023年4月30日前迁入）
- ⚠️ **五年一户**: 从2021年起计算，须满五年
- ⚠️ **预警学校**: 红色预警学校增至12所小学、4所初中

### 报名注意事项
- ⚠️ 4月27日后不可更改入学信息
- ⚠️ 公办民办只能选一个报名
- ⚠️ 民办未录取回对口统筹
- ⚠️ 第一批验证未通过参加第二批验证

### 材料准备
- ⚠️ 户口本原件及复印件
- ⚠️ 房产证原件及复印件
- ⚠️ 出生证明原件及复印件
- ⚠️ 预防接种卡原件及复印件
