## 2026年浦东新区招生时间线

| 时间 | 事项 | 备注 |
|------|------|------|
| 3月 | 公布招生入学政策 | 关注五年一户、入户年限 |
| 4月上旬 | 公布招生简章和学区划分 | 确认对口小区 |
| 4月中旬 | 校园开放日 | 部分学校需预约 |
| 4月下旬 | 入学信息登记 | 在就读幼儿园/指定地点 |
| 5月 | 公办小学第一批验证 | 人户一致学生验证 |
| 5月中 | 民办小学摇号 | 如报名民办 |
| 6月 | 公办小学第二批验证 | 统筹安排 |
| 8月 | 发放入学通知书 | 完成招生 |

## 家长行动清单

- 2025年底前：确认入户年限是否达标
- 2026年3月：关注政策发布
- 2026年4月：确认对口信息
- 2026年4月：参加校园开放日
