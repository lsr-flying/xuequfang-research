## 嘉定区民办学校汇总

### 桃李园实验学校

- **类型**: 九年一贯制
- **小学部**: 有
- **初中部**: 有
- **建校时间**: 2002年
- **学费**: 2-3万/学期
- **特点**: 嘉定民办最强，中考市重率60%+
- **招生方式**: 统招+直升
- **政府购买学位**: 是

### 民办华二初级中学

- **类型**: 初中
- **建校时间**: 2010年
- **学费**: 较高
- **特点**: 华二系，竞赛强，四校八大30%
- **招生方式**: 统招
- **政府购买学位**: 否

### 嘉定世外学校

- **类型**: 九年一贯制
- **小学部**: 有
- **初中部**: 有
- **建校时间**: 2018年
- **学费**: 较高
- **特点**: 世外系，国际化路线
- **招生方式**: 统招+直升
- **政府购买学位**: 否

### 华师大双语学校

- **类型**: 十二年一贯制
- **小学部**: 有
- **初中部**: 有
- **高中部**: 有
- **建校时间**: 2015年
- **学费**: 较高
- **特点**: K-16双语教育
- **招生方式**: 统招+直升
- **政府购买学位**: 否

### 远东学校

- **类型**: 九年一贯制
- **小学部**: 有
- **初中部**: 有
- **建校时间**: 2002年
- **学费**: 2-3万/学期
- **特点**: 性价比高
- **招生方式**: 统招+直升
- **政府购买学位**: 是

## 公民同招下的策略

1. **冲民办**: 公办保底，民办冲高
2. **选民办校**: 九年一贯制更稳（直升不受摇号）
3. **看政府购买学位**: 影响学费和招生方式
