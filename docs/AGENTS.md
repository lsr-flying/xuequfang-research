这是一个用于研究上海学区房的Obsidian知识库，重点关注嘉定区，帮助为2021年9月后入学的孩子挑选合适的学区房。项目已完成，包含完整的学校信息、房产分析、政策文件等。

## 代码/文档规范

### 信息来源优先级
1. **官方文件** - 教育局官网、政府公告（最权威）
2. **官方解读** - 官方媒体的政策解读
3. **家长经验** - 真实案例（需标注来源）
4. **中介信息** - 链家等（参考，需交叉验证）

### 每条信息需标注
```yaml
updated: YYYY-MM
source: 来源
reliability: 高/中/低
```

### 研究原则
1. **时效性优先** - 政策每年变，必须用最新年份数据（2025年）
2. **双向验证** - 重要信息至少两个来源确认
3. **标注时间** - 任何预测都要标注基于哪年数据
4. **保留原文** - 关键政策要保留原文引用
5. **彻底完成** - 不留空白目录，所有待办都要自觉完成
