# 上海嘉定区学区房研究项目

> 为2021年9月后入学选择嘉定区最佳学区房的升学置业知识库

## 项目概述

这是一个用于研究上海学区房的Obsidian知识库，重点关注嘉定区，帮助为孩子挑选合适的学区房。**项目已完善**，包含政策文件、学校信息、房产数据、入学路径分析等完整内容。

## 核心内容

### 政策文件
- 2024/2025年嘉定区招生实施意见
- 招生预警信息（16所学校红色预警）
- 入学排序办法
- 关键时间节点（2024/2025）

### 学校信息（~110+文件）
- **第一梯队小学**: 普通小学、疁城实验、嘉定实验、中科院、同济附属、新城实验、德富路、宋校嘉定、交大附中实验、上师大附属嘉定
- **第二梯队小学**: 城中路、南翔、苏民、古猗、留云、嘉一附小、马陆
- **第三梯队小学**: 安亭、方泰、黄渡、江桥、封浜、华江、外冈、徐行、曹王等
- **第一梯队初中**: 嘉定二中、嘉定一中、交大附中嘉定分校、新城实验中学、中科院实验、嘉一实验等
- **第二/三梯队初中**: 启良、南翔、马陆育才、震川、同济附中、江桥实验等
- **民办学校**: 桃李园、华二初级、嘉定世外、华师大双语、远东

### 房产数据（10个板块）
- 嘉定镇板块、嘉定新城板块、南翔板块、菊园新区板块、马陆板块
- 安亭板块、江桥板块、外冈板块、徐行板块、真新板块
- 含价格走势、成交量、地铁配套、商业配套

### 入学路径
- 优质双学区路径
- 潜力新校路径
- 民办路线
- 梯度择校

### 规划学校（5所）
- 上海大学附属嘉定实验学校
- 上海市实验学校嘉定新城分校
- 上海交通大学附属嘉定实验学校
- 嘉一实验初级中学
- 菊园新区南片规划学校
